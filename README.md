# Telegram-бот: опитувальник про дизайн квартири

Простий бот, що збирає ім'я/прізвище та відповіді на опитування про дизайн
квартири й надсилає заявку власнику в Telegram.

## Кроки для запуску

### 1. Створи бота в Telegram
1. Відкрий Telegram, знайди **@BotFather**.
2. Надішли `/newbot`, дай ім'я та юзернейм (має закінчуватись на `bot`).
3. BotFather видасть **токен** виду `123456:ABC-DEF...`. Скопіюй його.

### 2. Налаштуй файл `.env`
1. Скопіюй `.env.example` у `.env`.
2. Встав токен у `BOT_TOKEN=`.
3. `OWNER_CHAT_ID` поки залиш порожнім.

### 3. Дізнайся свій chat_id
1. Запусти бота (див. крок 4).
2. У Telegram напиши своєму боту команду `/whoami`.
3. Він відповість числом — це твій `chat_id`. Встав його у `.env`
   (`OWNER_CHAT_ID=...`) і перезапусти бота.

### 4. Запуск
```bash
pip install -r requirements.txt
python bot.py
```

## Команди бота
- `/start` — почати опитування
- `/cancel` — скасувати
- `/whoami` — показати свій chat_id
